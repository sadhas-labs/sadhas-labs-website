Welcome to the repository for the **Sadhas Labs Website**, a modern and responsive web presence for a non-profit organization dedicated to integrating Carnatic music with machine learning and data science. This project is built using React and modern CSS techniques to deliver a sleek, dynamic, and interactive user experience.

## Overview

**Sadhas Labs** is a non-profit organization that aims to bring state-of-the-art data technologies to further the knowledge of Carnatic music. This website provides an engaging interface that includes dynamic text animations and interactive team member flip cards, all wrapped in a modern design.

---

## Features

- **Responsive Design:** Adapts seamlessly to various screen sizes.
- **Dynamic Tagline:** Features a rotating word in the tagline (e.g., innovation, technology, research, AI, ML) with a smooth fade effect.
- **Interactive Team Section:** Flip card design for team members that reveals additional information on hover.
- **Modern Aesthetics:** Utilizes CSS Grid, Flexbox, and subtle animations for a clean and engaging look.
- **Easy Customization:** Modular code structure that makes it simple to update content and styles.

---

## Tech Stack

- **React:** For building the user interface.
- **CSS:** For styling, including modern layout techniques (Flexbox and Grid).
- **JavaScript (ES6+):** Leveraging modern features to enhance interactivity.
